# О проекте


Веб-приложение для управления персоналом в организациях. Система позволяет HR-специалистам вести учет сотрудников, отделов, должностей и кадровых операций.


## Основные возможности

- Управление организациями, отделами, должностями.
- Ведение личных дел сотрудников с историей трудоустройства.
- Хранение сканов документов (паспорт, трудовая книжка, дипломы).
- Регистрация HR-операций (приём, перевод, увольнение).
- Аудит всех действий в системе.
- Ролевая модель доступа (администратор, HR-специалист).


## Stack

- **ОС:** Ubuntu 24.04
- **IDE:** VS Code + DBeaver
- **База данных:** PostgreSQL 17 (Docker)
- **Контейнеризация:** Docker + Docker Compose
- **Веб-сервер:** Nginx
- **Версионный контроль:** Git / GitHub
