# Руководство пользователя

## Вход в систему

1. Откройте в браузере адрес вашего экземпляра Areal HR.
2. На странице входа введите **логин** и **пароль**, выданные администратором.
3. После успешной аутентификации вы попадёте на главную страницу с меню навигации.

## Обзор интерфейса

В верхней части экрана расположена навигационная панель со следующими пунктами:

- **Организации** — управление юридическими лицами.
- **Отделы** — управление структурными подразделениями.
- **Должности** — управление штатными позициями.
- **Сотрудники** — управление личными делами сотрудников.

В правом углу находится кнопка **«Выйти»** для завершения сессии.


## Организации

Организация — это юридическое лицо (компания, филиал), в рамках которого ведётся кадровый учёт.

### Просмотр списка организаций

1. Перейдите в раздел **«Организации»**.
2. На экране отобразится таблица со всеми организациями, зарегистрированными в системе.
3. Для каждой организации показаны:
   - Название организации
   - Комментарий
   - Дата создания

### Добавление новой организации

1. Нажмите кнопку **«Добавить организацию»**.
2. Заполните форму:
   - **Название** — полное наименование юридического лица.
   - **Комментарий** — (опционально).
3. Нажмите кнопку **«Создать»**, для сохранения или кнопку **«Отмена»**, для отмены операции.

### Редактирование организации

1. В таблице организаций на строке с нужной организацией нажмите на кнопку **«Редактировать»**.
2. Внесите необходимые правки в форму.
3. Сохраните изменения нажатием на кнопку **«Сохранить»** или нажмите на кнопку **«Отмена»**, для отмены операции.

### Удаление организации

1. В таблице организаций на строке с нужной организацией нажмите на кнопку **«Удалить»**.
2. Подтвердите действие во всплывающем окне нажатием на кнопку **«Удалить»** или нажмите на кнопку **«Отмена»**, для отмены операции.


## Отделы

Отдел — это структурное подразделение внутри организации (например, «Отдел разработки», «Бухгалтерия»).
Система поддерживает иерархическую (вложенную) структуру отделов.

### Просмотр списка отделов

1. Перейдите в раздел **«Отделы»**.
2. На экране отобразится таблица со всеми отделами, зарегистрированными в системе.
3. Для каждого отдела показаны:
   - Название организации, к которой относится отдел
   - Название отдела
   - Родительский отдел
   - Комментарий

### Добавление отдела

1. Нажмите кнопку **«Добавить отдел»**.
2. Заполните форму:
   - **Организация** — выберите организацию из выпадающего списка.
   - **Название** — например, «Отдел кадров».
   - **Родительский отдел** — выберите отдел из выпадающего списка (опционально).
   - **Комментарий** — (опционально).
   
3. Нажмите кнопку **«Создать»**, для сохранения или кнопку **«Отмена»**, для отмены операции.

### Редактирование и удаление

Аналогично организациям: выберите отдел, внесите изменения или удалите.


## Должности

Должность — название роли сотрудника в компании.

### Просмотр списка должностей

1. Перейдите в раздел **«Должности»**.
2. На экране отобразится таблица со всеми должностями, зарегистрированными в системе.
3. Для каждой должности показаны:
   - Название должности
   - Дата создания

### Добавление должности

1. Нажмите кнопку **«Добавить должность»**.
2. Заполните форму:
   - **Название** — например, «Full Stack Разработчик».

3. Нажмите кнопку **«Создать»**, для сохранения или кнопку **«Отмена»**, для отмены операции.

### Редактирование и удаление

Аналогично организациям: выберите должность, внесите изменения или удалите.


## Сотрудники

Сотрудник — это физическое лицо, работающее в организации. Для каждого сотрудника ведётся личное дело с персональными данными, историей трудоустройства, документами и HR-операциями.

### Просмотр списка сотрудников

1. Перейдите в раздел **«Сотрудники»**.
2. На экране отобразится таблица со всеми сотрудниками, зарегистрированными в системе.
3. Для каждого сотрудника показаны:
   - ФИО
   - Статус
   - Отдел
   - Должность
4. Каждая строка таблицы кликабельна, при нажатии открывается карточка сотрудника с подробными данными:
      - **Основные данные**
         - Фамилия
         - Имя
         - Отчество
         - Дата рождения
      - **Паспортные данные**
         - Серия
         - Номер
         - Код подразделения
         - Дата выдачи
         - Кем выдан
      - **Адрес регистрации**
         - Область
         - Населенный пункт
         - Улица
         - Дом
         - Клрпус
         - Квартира
      - **Документы сотрудника**
         - Название документа
         - Дата загрузки
      - **История кадровых операций**
         - Тип операции
         - Отдел
         - Должность
         - Зарплата
         - Дата операции                  

### Добавление сотрудника

1. Нажмите кнопку **«Добавить сотрудника»**.
2. Заполните форму:
   - **Фамилие** — например, «Иванов».
   - **Имя** — например, «Иван».
   - **Отчество** — например, «Иванович».
   - **Дата рождения** — например, «22.07.1986».

3. Нажмите кнопку **«Создать»**, для сохранения или кнопку **«Отмена»**, для отмены операции.
4. После нажатия кнопки **«Создать»** происходит автоматический переход в карточку сотрудника.
5. В карточке сотрудника для заполнения личных данных нажмите кнопку **«Редактировать»** и в 
появившемся окне заполните формы, которые переключаются следующими вкладками:
   - **Основные**
      - Фамилия (уже заполнено)
      - Имя (уже заполнено)
      - Отчество (уже заполнено)
      - Дата рождения (уже заполнено)
   - **Паспорт**
      - Серия паспорта
      - Номер паспорта
      - Дата выдачи
      - Код подразделения
      - Кем выдан
   - **Адрес**
      - Область
      - Населенный пункт
      - Улица
      - Дом
      - Корпус
      - Квартира      
6. Нажмите кнопку **«Сохранить»**, для сохранения или кнопку **«Отмена»**, для отмены операции.    

## Документы

Документы — система позволяет хранить сканы документов сотрудников

### Просмотр списка документов

1. В карточке сотрудника перейдите на вкладку **«Документы»**.
2. На экране отобразится таблица со всеми документами, зарегистрированными в системе.
3. Для каждого документа показаны:
   - Название документа
   - Дата загрузки
4. Для скачивания в таблице на строке нужного документа нажмите кнопку **«Скачать»**.

### Загрузка документа

1. Нажмите кнопку **«Загрузить документ»**.
2. Заполните форму:
   - **Название документа** — например, «Паспорт».
   - **Обзор** — укажите местонахождение документа на жестком диске через проводник.
3. Нажмите кнопку **«Загрузить»**, для сохранения или кнопку **«Отмена»**, для отмены операции.

### Удаление документа

Аналогично организациям: выберите документ и удалите.

## Кадровые операции

Кадровые операции - система позволяет проводить следующие операции:
   - **Прием на работу** — операция является первой, после добавления нового сотрудника.
   Она присваивает отдел, должность и заработную плату сотруднику.
   - **Перевод** — операция, для перевода сотрудника в другой отдел в рамках одной организации.
   - **Изменение зарплаты** —  изменение заработной платы сотрудникаю.
   - **Увольнение** —  увольнение сотрудника.


### Просмотр списка кадровых операций

1. В карточке сотрудника перейдите на вкладку **«Кадровые операции»**.
2. На экране отобразится таблица со всеми операциями, зарегистрированными в системе.
3. Для каждой операции показаны:
   - Тип операции
   - Отдел
   - Должность
   - Зарплата
   - Дата операции

### Создание кадровой операции

1. Нажмите кнопку **«Новая операция»**.
2. В появившейся форме выберите **«Тип операции»** из выпадающего списка.
3. Заполните форму в зависимости от операции:
   - **Прием на работу**
      - Отдел
      - Должность
      - Зарплата
   - **Перевод**
      - Отдел
   - **Изменение зарплаты**
      - Зарплата
   - **Увольнение**
      - Не требуется
4. Нажмите кнопку **«Создать»**, для сохранения или кнопку **«Отмена»**, для отмены операции.

### Редактирование и удаление кадровой операции

Все кадровые операции остаются в системе неизменными для аудита. Чтобы исправить ошибку, создайте новую операцию с верными данными — обе записи сохранятся в истории.
